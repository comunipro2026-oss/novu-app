# NOVU — El chat que habla el idioma de tu cerebro

Primera plataforma de mensajería adaptativa con IA para personas neurodivergentes.

## 🚀 Ver la app en vivo

**[novu-app.github.io](https://TU-USUARIO.github.io/novu-app)**

## ✨ Funciones

- 🎙️ **Audio Inteligente** — Resume audios largos en segundos
- 💜 **Detector de Tono** — Detecta mensajes agresivos antes de enviar
- 🧩 **Modo Neurodivergente** — Pictogramas y comunicación adaptativa
- ⚡ **Chat por Intención** — Organizado por objetivo, no por conversación

## 🛠️ Tecnología

- HTML + CSS + JavaScript vanilla
- Sin dependencias externas
- Funciona en cualquier navegador

## 📦 Deploy

Publicado con GitHub Pages. Cualquier push a `main` actualiza la app automáticamente.
